VERIFICATION

Verification is intended to assist the Chocolatey moderators and community
in verifying that this package's contents are trustworthy.

This package does not embed the gocore binary. At install time,
chocolateyinstall.ps1 downloads the official Windows release from:

  https://github.com/core-coin/go-core/releases/download/v2.2.2/gocore-windows-x86_64.exe

and verifies its SHA-256 against the value embedded in the script:

  a0e1f0bea509bc4b6df3c6468b698ad0e74a4113344efb0b238ca3a394a9c831

The same checksum is published by the upstream release as:

  https://github.com/core-coin/go-core/releases/download/v2.2.2/gocore-windows-x86_64.exe.checksum

To verify manually, download both files and compare the SHA-256 of the .exe
to the value in the .checksum file and to the value above.
